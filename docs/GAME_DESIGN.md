# 게임 기획서 — 이소 스테이지 기반 스토리 RPG

## 1. 게임 개요

2D 탑다운 방식의 스테이지 기반 RPG. 스토리 흐름에 따라 챕터(스테이지)가 구분되며, 유저는 **[중간 맵(거점)]** 과 **[챕터(스토리 월드)]** 를 오가며 캐릭터를 성장시키고 엔딩을 향해 나아간다.

---

## 2. 월드 구조

### 2.1. 중간 맵 (거점 / Base Camp)

모든 챕터의 시작점이자 종착점인 허브(Hub) 공간.

| 항목 | 내용 |
|------|------|
| 역할 | 정비, 강화, 스펙업, 스테이지 선택 |
| 안전지대 | 적 없음, 데이터 영구 보존 |
| 주요 NPC | 대장장이, 상점, 창고 등 |
| 특수 콘텐츠 | 외전 (곁줄기 스토리 + 파밍 인스턴스) |

### 2.2. 챕터 (스테이지 / Chapter)

하나의 에피소드가 진행되는 독립된 인스턴스 월드.

```
챕터 구조:
    [중심 마을] ──── [필드 A]
         │      ├── [필드 B] ── [던전 / 보스]
         │      └── [필드 C]
         └── NPC(퀘스트), 잡화 상점 (몬스터 없음)
```

**진행 흐름:**
1. 마을에서 퀘스트 수주
2. 필드 이동 및 사냥
3. 마을 복귀(퀘스트 보고)
4. 다음 필드 해금
5. 최종 보스 처치 → 챕터 완료

---

## 3. 캐릭터 성장 시스템

### 3.1. 레벨 & 스탯

- **경험치 획득:** 몬스터 처치, 퀘스트 완료
- **스탯 자동 상승:** 레벨업 시 정해진 테이블에 따라 자동 상승 (유저 직접 분배 없음)
- **전투력 구성:** 기본 스탯(레벨) + 장비 능력치 + 장비 강화 수치

### 3.2. 스킬 시스템

| 항목 | 내용 |
|------|------|
| 획득 방식 | 각 챕터 최종 보스 처치 시 보스 고유 스킬 해금 |
| 예시 | 1챕터 보스 처치 → '돌진' / 2챕터 보스 처치 → '암살' |
| 사용 | 해금 즉시 장착 가능 |
| 성장 | 스킬 레벨업 없음 (향후 업데이트 검토 예정) |

### 3.3. 장비 강화

| 항목 | 내용 |
|------|------|
| 장소 | 중간 맵 대장장이 NPC 한정 |
| 재료 | 골드(Gold)만 소모 — 강화석 등 복잡한 재료 없음 |
| 구조 | 장비 + 골드 → 성공 시 능력치 상승 |
| 의도 | 파밍 스트레스 감소, 골드의 가치 상승 |

---

## 4. 저장 및 초기화 시스템 (핵심 로직)

### 4.1. 기본 저장 (챕터 내 플레이)

챕터 내부에서 플레이하는 동안은 **모든 데이터 실시간 저장**.

- 캐릭터 위치, 경험치, 획득 아이템, 메인/서브 퀘스트 진행 단계
- 강제 종료 / 앱 재시작 시 마지막 상태에서 이어하기 지원

### 4.2. 중간 맵 복귀 시 초기화 (전략적 후퇴)

유저가 스펙업을 위해 중간 맵으로 후퇴를 선택했을 때 적용.

| 항목 | 처리 |
|------|------|
| 유지 | 경험치, 레벨, 아이템(전리품), 골드 — **100% 보존** |
| 초기화 | 해당 챕터의 메인/서브 퀘스트 진행도, 맵 탐험 상태, 챕터 내 위치 |

**목적:** 외전에서 골드/경험치를 벌어 스펙을 올린 뒤 챕터를 처음부터 재도전.

## 5. 중간 맵 주요 콘텐츠

### 5.1. 외전 (곁줄기 스토리 + 파밍 콘텐츠)

메인 스토리(이소 시점)와 별개로 진행되는 곁줄기 스토리 콘텐츠.
다른 인물의 시각에서 세계관을 보여주는 서사 장치이자, 챕터 공략을 위한 전투력 상승 수단.

**핵심 특징:**
- 플레이어가 **이소가 아닌 다른 인물**을 직접 조작
- 해당 인물의 시점에서 진행되는 독립된 미니 스토리
- 세션 단위로 진행 (입장 → 미션 수행 → 베이스캠프 복귀)

**진행 방식:**
- 중간 맵에서 외전 퀘스트 수주
- 별도 인스턴스 맵에서 해당 인물로 전환하여 시작
- 미션 목표 달성 → 세션 종료 → 보상 획득 → 베이스캠프 복귀

**미션 유형:**

| 유형 | 설명 |
|------|------|
| 처치 | 제한 시간 내 특정 타겟(우두머리) 찾아 처치 |
| 생존 | 쏟아지는 적 공격 속에서 제한 시간 동안 생존 |
| 탐색 | 적의 방해를 뚫고 제한 시간 내 특정 아이템 확보 |

**보상:**
- 일반 챕터 필드 대비 높은 드랍률, 골드 획득량
- 미션 완료 시 다량의 경험치 + 확정 보상

**스크립팅:** 외전 미션은 @quest 블록으로 정의. 파형 소환이 필요한 생존 미션에 한해 @wave 병용.

### 5.2. 상점 및 정비 시설

| 시설 | 기능 |
|------|------|
| 대장간 | 골드 소모 → 장비 강화 |
| 무기/방어구 상점 | 기본 장비 구매/판매 |
| 창고 | 인벤토리 정리 |

---

## 6. 이동 및 편의 기능

### 6.1. 이동 방식

- **도보 이동:** 필드 간 연결 통로를 통해 이동
- **마을 귀환 주문서:** 필드 사냥 중 사용 시 해당 챕터 중심 마을로 즉시 텔레포트 (별도 숏컷 없음)

### 6.2. 맵 UI

| UI | 내용 |
|----|------|
| 미니맵 (HUD) | 화면 구석, 지형/내 위치/몬스터·NPC 위치 표시 |
| 전체맵 (World Map) | 메뉴 호출, 챕터 전체 구조 + 현재 위치 + 퀘스트 마커 |

## 7. 스테이지 진입 및 완료 프로세스

### 7.1. 챕터 진입 (권장 전투력 체크)

```
중간 맵에서 챕터 선택
    ↓
스펙 체크
    ├── [미달] → 경고 팝업 "공략이 매우 어렵습니다. 입장하시겠습니까?"
    │           ├── [입장] → 강행 입장
    │           └── [취소] → 중간 맵 유지
    └── [충족] → 즉시 입장
```

### 7.2. 챕터 진행 중

```
마을 ↔ 필드 (퀘스트 수행)
         ↓ 도저히 클리어 불가
    메뉴 → [중간 맵으로 후퇴]
         ↓ 퀘스트 리셋 경고 확인
    중간 맵 → 스펙업 → 재도전
```

### 7.3. 챕터 완료

```
최종 보스 처치 + 메인 퀘스트 완료
    ↓
1. 보스 고유 스킬 획득
2. 중심 마을에 [다음 챕터 / 중간 맵] 포탈 생성
3. 미완료 서브 퀘스트 있을 시 → 포탈 진입 시 Alert
    ↓
포탈 진입 → 다음 챕터 or 중간 맵
```
